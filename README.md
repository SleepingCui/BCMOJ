### **BCMOJ** 是一个轻量化的在线代码评测系统。  
#### 前端基于 **HTML + JavaScript**，后端使用 **Flask** 提供接口服务，评测核心由 **Java** 实现，使用 **MySQL** 存储数据。

---

## 架构

```plaintext
前端 (HTML + JavaScript)
        │
        ▼
Flask Web Server ─────▶ MySQL 数据库
        │
        ▼
  判题服务器 (Java)
```
